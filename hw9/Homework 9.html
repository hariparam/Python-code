Perceptrons

In this section, you will implement two varieties of the standard perceptron: one which performs binary classification, distinguishing between positive and negative instances, and one which performs multiclass classification, distinguishing between an arbitrary number of labeled groups.

As in previous assignments, your use of external code should be limited to built-in Python modules, which excludes packages such as NumPy and NLTK.

  1. A binary perceptron is one of the simplest examples of a linear classifier. Given a set of data points each associated with a positive or negative label, the goal is to learn a vector w⃗ w→ such that w⃗ ⋅x⃗ +>0w→⋅x→+>0 for positive instances x⃗ +x→+ and w⃗ ⋅x⃗ −≤0w→⋅x→−≤0 for negative instances x⃗ −x→−.

    One learning algorithm for this problem initializes the weight vector w⃗ w→ to the zero vector, then loops through the training data for a fixed number of iterations, adjusting the weight vector whenever a sample is misclassified.

    Input:

    A list TT of training examples (x⃗ 1,y1),⋯,(x⃗ n,yn)(x→1,y1),⋯,(x→n,yn), where yi∈{+,−}yi∈{+,−}; the number of passes NN to make over the data set.

    Output:

    The weight vector w⃗ w→.

    Procedure:

    1. Initialize the weight vector as w⃗ ←0w→←0.
    2. for iteration =1=1 to NN do
    3. for each example (x⃗ i,yi)∈T(x→i,yi)∈T do
    4. Compute the predicted class as y^i=sign(w⃗ ⋅x⃗ i)y^i=sign(w→⋅x→i).
    5. if y^i≠yiy^i≠yi then
    6. Set w⃗ ←w⃗ +x⃗ iw→←w→+x→i if yiyi is positive, or w⃗ ←w⃗ −x⃗ iw→←w→−x→i if yiyi is negative.
    7. end if
    8. end for
    9. end for

    During initialization, you should train the weight vector w⃗ w→ on the input data using iterations passes over the data set, then store w⃗ w→ as an internal variable for future use. Each example in the examples list will be a 22-tuple (x⃗ x→, yy) of a data point paired with its binary label True or False.

    The prediction method should take as input an unlabeled example x⃗ x→ and compute the predicted label as sign(w⃗ ⋅x⃗ )sign(w→⋅x→), returning True if w⃗ ⋅x⃗ >0w→⋅x→>0 or False if w⃗ ⋅x⃗ ≤0w→⋅x→≤0.

    In this assignment, we will represent vectors such as w⃗ w→ and x⃗ x→ in Python as dictionary mappings from feature names to values. If a feature is absent from a vector, then its value is assumed to be zero. This allows for efficient storage of sparse, high-dimensional vectors, and encourages efficient implementations which consider only non-zero elements when computing dot products.   2. A multiclass perceptron uses the same linear classification framework as a binary perceptron, but can accommodate an arbitrary number of classes rather than just two. Given a set of data points and associated labels, where the labels are assumed to be drawn from some set {ℓ1,⋯,ℓm}{ℓ1,⋯,ℓm}, the goal is to learn a collection of weight vectors w⃗ ℓ1,⋯,w⃗ ℓmw→ℓ1,⋯,w→ℓm such that argmaxℓk(w⃗ ℓk⋅x⃗ )argmaxℓk(w→ℓk⋅x→) equals the correct label ℓℓ for each input pair (x⃗ ,ℓ)(x→,ℓ).

    The learning algorithm for this problem is similar to the one for the binary case. All weight vectors are first initialized to zero vectors, and then several passes are made over the training data, with the appropriate weight vectors being adjusted whenever a sample is misclassified.

    Input:

    A list TT of training examples (x⃗ 1,y1),⋯,(x⃗ n,yn)(x→1,y1),⋯,(x→n,yn), where the labels yiyi are drawn from the set {ℓ1,⋯,ℓm}{ℓ1,⋯,ℓm}; the number of passes NN to make over the data set.

    Output:

    The weight vectors w⃗ ℓ1,⋯,w⃗ ℓmw→ℓ1,⋯,w→ℓm.

    Procedure:

    1. Initialize the weight vectors as w⃗ ℓk←0w→ℓk←0 for k=1,⋯,mk=1,⋯,m.
    2. for iteration =1=1 to NN do
    3. for each example (x⃗ i,yi)∈T(x→i,yi)∈T do
    4. Compute the predicted label as y^i=argmaxℓk(w⃗ ℓk⋅x⃗ i)y^i=argmaxℓk(w→ℓk⋅x→i).
    5. if y^i≠yiy^i≠yi then
    6. Increase the score for the correct class by setting w⃗ yi←w⃗ yi+x⃗ iw→yi←w→yi+x→i.
    7. Decrease the score for the predicted class by setting w⃗ y^i←w⃗ y^i−x⃗ iw→y^i←w→y^i−x→i.
    8. end if
    9. end for
    10. end for

  3. A simple data set of one-dimensional data is given in data.bias, where each example consists of a single positive real-valued feature paired with a binary label. Because the binary perceptron discussed in the previous section contains no bias term, a classifier will not be able to directly distinguish between the two classes of points, despite them being linearly separable. To see why, we observe that if the weight vector (consisting of a single component) is positive, then all instances will be labeled as positive, and if the weight vector is negative, then all instances will be labeled as negative. It is therefore necessary to augment the input data with an additional feature in order to allow a constant bias term to be learned.

    Your task is to implement the __init__(self, data) and classify(self, instance) methods in the BiasClassifier class to perform training and classification on this data set. Example data will be provided as a list of 22-tuples (x⃗ x→, yy) of real numbers paired with binary labels. Test instances will be single numbers, and should be classified as either True or False. As discussed above, instances will have to be augmented with an additional feature before being fed into a perceptron in order for proper learning and classification to take place.

    >>> c = BiasClassifier(data.bias)
>>> [c.classify(x) for x in (-1, 0, 0.5, 1.5, 2)]
[False, False, False, True, True]
